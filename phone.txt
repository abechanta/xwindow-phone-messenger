１．ファイル
    phone は以下のファイルで構成されます．

	・phone.c	phone のソースプログラム．Ｃ言語で記述されている．
	・phone.h	phone.c のヘッダーファイル．
	・phone.txt	このドキュメントファイル．
	・~/.phone	別名登録ファイル．

２．起動方法

    コマンドライン上で

	% phone unix0:0 [unix1:0] [unix2:0] [unix3:0]

  または

	% phone unix0 [unix1] [unix2] [unix3]

  と入力することで命令が実行されます．引数として会話したいホスト名を４つまで
  入力します．ホスト名に付随するディスプレイ番号やスクリーン番号は省略できま
  す．
    また， .phone に別名登録してあるホスト名はその別名を使ってディスプレイを
  指定できます．例えば .phone（別名登録ファイル） に

	NAME	a	longusername	unix0:0
	 ^	 ^	 ^	 ^
	 |	 |	 |	 |
       別名宣言 別名  ユーザ名  ホスト名

  とある場合には，

	% phone unix0:0

  とするかわりに

	% phone a

  と命令することができます．

３．実行

    まず実行前に，相手側の端末で

	% xhost +

  が行われていることを確かめてください．このコマンドは他のホストからのアクセ
  スを許可するもので，このコマンドを実行していない場合には相手側ホストとのア
  クセスに失敗することがあります．
    命令を実行した端末（send）とディスプレイオープンに成功した端末（recieve）
  には，ウインドウが表示されます．
    send ウインドウにマウスカーソルを持っていき，キーボードでタイプすることで
  メッセージを送ることができます．同様に recieve ウインドウにマウスカーソルを
  持っていき，キーボードでタイプすることで相手側にメッセージを送ることができ
  ます．
    send ウインドウと recieve ウインドウには同じものが表示されています．
    メッセージは英字入力がデフォルトになっていますが，各自のウインドウでマウ
  スボタンの１（左）を押すことで仮名（ローマ字）入力に切り替えることができま
  す．ローマ字入力モードになると情報表示欄に [] が出現し現在のキーバッファを
  確認しながらの入力ができます．
    メッセージの送信を終了したいときには，マウスボタンの３（右）を押してくだ
  さい．

４．補足

	・.phone（別名登録ファイル）の書式について

	    .phone は別名登録を行うファイルで，通常のテキストエディタで編集す
	  ることができます．このファイルは各ユーザのホームディレクトリに置い
	  ておいてください．以下に宣言を紹介します．

	    ＜NAME＞
		説明
		    phone コマンドによって使うことのできる別名を登録する．
		書式
			NAME	a	longusername	unix0:0
			 ^	 ^	 ^	 ^
			 |	 |	 |	 |
		       別名宣言 別名  ユーザ名  ホスト名

	    ＜REM＞
		説明
		    このコマンドから行末までをコメントとして扱う．
		書式
			REM	this line is comment.

	    ＜END＞
		説明
		    別名登録ファイルの終端を表す．
		書式
			END

	・stdout の出力について

	    たまにメッセージ送信中にコマンドラインへの出力が起こりますが，
	  とくに深い意味はないのでご了承下さい．
